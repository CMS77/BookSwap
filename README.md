## 📖 Introduction  
This project consists of the development of an **online platform for lending books between users**.  
The main idea is to allow individuals to share books they own with other interested users, promoting **access to reading** and encouraging **collaboration within the community**.

The platform works as a **book-sharing network**, where each user can create their own **digital library**, make books available for lending, and request books they would like to read.

---

## 🎯 Platform Objectives  
The main objective of this platform is to **facilitate book lending between users**, providing the following features:

- 📌 Register and manage books available for lending  
- 🔎 Search for books available within the community  
- 📥 Request book loans from other users  
- 📊 Manage borrowed and lent books  
- 💬 Enable communication between users to arrange book delivery  

---

## 🤝 Expected Impact  
This platform aims to:

- 📚 Promote reading habits among users  
- 🤝 Encourage community collaboration and sharing  
- 💡 Provide a simple and accessible digital solution for managing book loans  
- ♻️ Support resource reuse by reducing the need to purchase new books
